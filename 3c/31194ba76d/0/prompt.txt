# UI Revamp: Soft/Organic Dark Mode

## Context

The Data Mapper app currently uses a cold, technical aesthetic (Geist Mono font, lime/green primary, 0.45rem radius, standard shadcn/ui look). The goal is a full visual revamp to a **Soft/Organic** aesthetic with **dark mode primary** — warm, inviting, rounded shapes, pastel gradients, gentle shadows, and smooth animations that make complex data mapping feel approachable.

---

## Design Direction

### Typography: **Sora Variable**
- Geometric sans-serif with distinctly rounded terminals — warm without being childish
- Headings: Sora SemiBold (600), `tracking-tight`
- Body/UI: Sora Regular (400) / Medium (500)
- Code contexts (mapping paths, Monaco): Keep Geist Mono Variable as secondary

### Color Palette (OKLCH, Dark Mode)

| Token | Value | Description |
|-------|-------|-------------|
| `--background` | `oklch(0.16 0.015 300)` | Deep plum-charcoal |
| `--foreground` | `oklch(0.93 0.01 80)` | Warm off-white |
| `--card` | `oklch(0.20 0.018 295)` | Elevated plum surface |
| `--primary` | `oklch(0.78 0.12 45)` | Soft peach/coral |
| `--secondary` | `oklch(0.72 0.10 290)` | Soft lavender |
| `--accent` | `oklch(0.75 0.10 175)` | Soft mint/teal |
| `--destructive` | `oklch(0.65 0.2 25)` | Soft warm red |
| `--muted` | `oklch(0.24 0.015 295)` | Subtle plum-grey |
| `--border` | `oklch(1 0 0 / 8%)` | Very subtle white |

New semantic tokens: `--source` (peach), `--target` (lavender), `--mapped` (rose), `--glass-bg`, `--glass-border`, `--connection-start/end`

### Border Radius: `0.875rem` base (up from `0.45rem`)
- Pill shapes (`rounded-full`) for tree nodes, mapping items, file badges, drag overlay

### Glass-morphism on cards
- `backdrop-blur-xl` + semi-transparent backgrounds + subtle borders + soft shadows

## Implementation Steps

### Step 1: Install font
```bash
bun add @fontsource-variable/sora
```

### Step 2: `src/styles.css` — Complete theme rewrite
- Add `@import "@fontsource-variable/sora"`

- Add new semantic variables (`--source`, `--target`, `--mapped`, `--glass-bg`, `--glass-border`, `--connection-start/end`)
